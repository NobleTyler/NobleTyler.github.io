**Eyes-free Error Correction for Touch Screen Devices:** This was a research project done in
conjuction with a proffessor for the purposes of testing Error Correction models using an eyes-free
interface. 
### What I learned from this project
* Working mainly on the research means you have to code fast and it can't break. 
* Being managed on a consistent basis allowed me to talk about my work frequently and gain insight into how to work effectively.
* Users will not always act how you expect and you have to be prepared for that.
