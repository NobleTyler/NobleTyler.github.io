**Portfolio:** This was a full-stack web application I created using Node/Express BootStrap MongoDb
and HTML. It was used to showcase some previous projects I had done. It was replaced by this one
after the original host of the Mongo cluster shutdown. 

### What I learned from this project
* Learning to use Mongo Express and Node was a very new experience for me
* Don't make a full-website when you could just make a static page.
* Cover up security vulnerabilities.
