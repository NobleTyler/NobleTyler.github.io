**SleepEKoala:** This was an Android app that was an alarm clock with a Koala theme! In the works it
was also able to implement a machine learning model in order to predict mood after sleeping. 

###  What I learned from this project
* Having previous Android experience came in handy for developing this app.
* Dealing with group dynamics and working alongside Machine Learning specialists was an excellent learning experience
* Using Android's timer API really simplified a lot of the code
